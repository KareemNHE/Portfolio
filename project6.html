Student Progression Tracker

Project Overview

This project was built to help education teams monitor student progress more effectively. It presented performance data in a clear visual format and supported staff decision making through alerts, administrative tools, and accessible records.

My Role

I contributed to the development of the application across frontend and backend areas, helping create a usable interface and structured data handling. The project strengthened my experience with Vue, PHP, SQLite, and user focused education technology.

Key Features

  • Student progress tracking with colour coded performance indicators.
  • Administrative tools for managing student data and viewing academic records.
  • Email alert functionality to support early intervention.
  • Dashboard style interface designed for quick interpretation by staff.

Technical Challenges

  • Presenting student progress data clearly without overwhelming users.
  • Designing workflows that supported both admin control and staff visibility.
  • Structuring SQLite data for reliable academic record handling.
