Project Overview

This project was designed as a complete online library system for managing digital and physical media rental workflows. It focused on secure user access, payment handling, database design, and a clear experience for both customers and administrators.

My Role

I contributed across the full development lifecycle, from requirements and database modelling to frontend implementation and secure backend functionality. The project strengthened my understanding of full-stack delivery, authentication, relational data design, and practical SDLC delivery.

Key Features

  • Membership registration and login with secure user access controls.
  • Media catalogue search and rental workflows for books, CDs, journals, and other resources.
  • Payment integration for rental processing and transaction flows.
  • Admin functionality for managing users, inventory, and availability.
  • Relational database design using MySQL and phpMyAdmin for structured data handling.

Technical Challenges

  • Designing a database structure that supported users, media items, rentals, and payments without duplication.
  • Ensuring the application remained easy to use while handling multiple user journeys.
  • Applying secure authentication and authorisation principles throughout the system.
