Full Stack Product Development

Plan B

A mobile-first platform helping users discover affordable and personalised date ideas based on their budget, location, occasion, and preferences.

Project Overview

Plan B was created to solve a common problem: finding meaningful and affordable things to do without spending hours searching online.

Users can build personalised plans by selecting criteria such as budget, occasion, duration, location, and preferred atmosphere. The platform then generates relevant recommendations while also allowing users to explore community-submitted ideas and experiences.

My Role

I designed and developed the platform independently, taking ownership of the full development lifecycle including product design, frontend development, backend architecture, database modelling, API integration, testing, and feature planning. The application is currently approaching public release.

Key Features

  • Personalised date recommendations based on budget, occasion, location, duration, and user preferences.
  • Community-driven suggestions where users can discover, discuss, and validate experiences.
  • Premium AI assistant capable of generating tailored date ideas and recommendations.
  • Mobile-first responsive design optimised for real-world use on smartphones.
  • User profiles, saved activities, and personalised recommendation history.
  • Scalable backend architecture supporting future feature expansion and user growth.

Technical Challenges

  • Designing recommendation logic capable of processing multiple user preferences simultaneously.
  • Creating an intuitive mobile-first experience while maintaining strong performance across devices.
  • Structuring data models for user-generated content and community engagement features.
  • Integrating AI-powered functionality while maintaining responsiveness and usability.
  • Building a scalable architecture capable of supporting future monetisation and premium features.

What This Project Demonstrates

  • Product-focused thinking with emphasis on user experience and commercial viability.
  • Mobile-first design principles and responsive web development.
  • Modern software engineering practices including scalability, maintainability, and modular design.
  • Independent project ownership from concept through development and release preparation.
