The Shot Sports Social Media App

A sports focused social media platform built with Flutter, Kotlin, Firebase, Provider, and MVVM architecture, with planned app store release.

Back to projects

Project Overview

The Shot is a mobile platform designed for sports fans, athletes, coaches, and beginners to share content, follow communities, and discuss sport in a focused digital space. The application combines social media interaction with sports specific feeds, discussion threads, real-time messaging, and personalised content.

My Role

I designed and developed the application end to end, covering mobile UI, architecture, Firebase backend integration, authentication, data modelling, state management, performance optimisation, and feature refinement. The project is ongoing and is being actively prepared for future release.

Key Features

  • Secure Firebase authentication with email verification and user access controls.
  • Real-time direct messaging and live content updates using Firebase services.
  • Personalised sports feeds using hashtags, following, interactions, and user behaviour.
  • Core social features including posts, likes, comments, sharing, blocking, private accounts, and profiles.
  • Discussion based threads, onboarding flows, dark mode, and app usage tracking for feed improvement.

Technical Challenges

  • Structuring Firebase data to support real-time updates while reducing unnecessary reads.
  • Maintaining responsive UI performance while handling dynamic feeds and messaging.
  • Keeping the codebase maintainable through MVVM architecture and Provider based state management.
