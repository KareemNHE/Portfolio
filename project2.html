Healthcare Navigation Platform

Chesterfield Royal Hospital Navigation System

A React based navigation system designed to help hospital visitors reach departments using QR codes, video walkthroughs, maps, and accessible directions.

Back to projects

Project Overview

This project addressed a real-world navigation challenge within a complex hospital environment. The platform allowed visitors to identify their current location using QR codes and receive clear route guidance through maps, video walkthroughs, audio, and written instructions.

My Role

I led the project team, coordinated Agile delivery, maintained communication with stakeholders, and guided the technical direction toward a React based frontend supported by scalable backend functionality. The project received highly positive stakeholder feedback due to its practical value and implementation potential.

Key Features

  • QR based location selection for visitors navigating from key points around the hospital.
  • Step by step route guidance using video walkthroughs, maps, audio, and subtitles.
  • Staff update capability so routes and content could be maintained without developer involvement.
  • Mobile responsive interface designed for visitors using the system on-site.
  • Accessible navigation experience with visual, written, and audio support.

Technical Challenges

  • Designing route guidance that was simple enough for stressed or unfamiliar visitors.
  • Creating a maintainable structure so hospital staff could update route content.
  • Balancing stakeholder requirements with technical feasibility during sprint delivery.
