Chesterfield Royal Hospital

About

Overview: A web-based navigation tool for Chesterfield Royal Hospital.Visitors can scan the QR codes dotted around the site and input their current location, then input their target destination. This will then display video walkthroughs with audio and subtitles, and an interactable maps to guide the visitors to their desired location. Designed for accessability and real-time route updates.

My Role

I served as the Team Lead, where I organised the Agile methodologies used, such as sprints, and kept close contact with the stake-holders. I diverted the development to use start using React.js to match the NHS's current development framework so that our website fits seamlessly and to use the best pratice possible. I coordinated with the back‑end team to design the JavaScript endpoints and oversaw usability testing with hospital staff.

Key Features & Technologies

  • React.js framework
  • QR Code Scanning & Processing
  • Step-by-Step Video Walkthroughs
  • Real-Time Route Updates via WebSockets
  • Mobile‑first Responsive Design
  • Google Maps Geolocation API
  • MySQL Database
